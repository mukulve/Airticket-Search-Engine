<h1>Air Ticket Search Engine</h1>

<h3>Scrapes Chepoair for all ticket data and displays them on webapp</h3>

<a>Technologies Used</a>
<ul>
  <li>Vue.Js</li>
  <li>Express.Js</li>
  <li>Puppeteer</li>
  <li>Node.Js</li>
</ul>

<a>Data Scraped from Chepoair</a>
<ul>
  <li>Airline</li>
  <li>Price</li>
  <li>Flight Duration</li>
</ul>
